About

Juggernaut - это набор независимых друг от друга компонентов, которые объеденены в единый модуль для CMS Битрикс. Вы можете использовать использовать компоненты по отдельности в любых своих проектах, либо использовать в совокупности для разработки под Битрик.

Get started

Установить модуль можно с помощью Compser: 

                      
                          composer require irpsv/juggernaut.bitrix
